# AgriKG for Nepal
A Knowledge Graph for Agricultural Decision Support in Nepal using Semantic Web Technologies, and GenerativeAI.

## Current Dataset Scope
### Crops:
  - Rice
  - Maize
  - Wheat
  - Millet
  - Barley

  ### Regions
    -Terai
    - Hill (Pahad)
    - Mountain (Himal)
  
  ### Disease
    - Relevant disease for each crop (not all but some major ones for now...)
    - Example: 
      - Rice Blast,
      - Rust, 
      - Powdery Millet, etc.
  
  ### Other Scopes
    - Climate
    - Soil
    - Treatment (disease specific)
    - Crop resilience traits
